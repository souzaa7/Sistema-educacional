# Sistema Educacional - Trabalho da Faculdade

Projeto desenvolvido por Gabriel Souza, Vitor Gabriel e João Vitor do primeiro semestre do curso de *Análise e Desenvolvimento de Sistemas*.

## Descrição do projeto

Este é um sistema educacional simples, desenvolvido em *C*, para fins acadêmicos.  
O sistema possui três tipos de usuários com diferentes permissões:

- *Administrador*
  - Cria usuários (professores e alunos)
  - Gerencia contas e acessos
- *Professor*
  - Posta material didático
  - Avalia atividades enviadas pelos alunos
- *Aluno*
  - Acessa material didático
  - Envia atividades solicitadas pelo professor
